Root Tower

Root Tower est un jeu de type rogue-lite où vous parcourez une mystérieuse tour remplie de créatures hostiles prêtes à tout pour vous empêcher d'en atteindre le sommet.Ce jeu est un projet que nous avons à faire en tant qu'étudiants SUP à EPITA. Le but du projet est de réaliser un jeu vidéo à l'aide de Unity en C#.
